# دفترچه — پروژه وبلاگ ساده با PHP

## راه‌اندازی

1. پوشهٔ `simple-blog` را داخل پوشهٔ سرور لوکال خود بگذارید (مثلاً `htdocs` در XAMPP).
2. فایل `database.sql` را در phpMyAdmin، از تب **Import** وارد کنید (دیتابیس `simple_blog` به‌صورت خودکار ساخته می‌شود).
3. در صورت نیاز، تنظیمات اتصال را در `config/db.php` ویرایش کنید (پیش‌فرض: `root` بدون رمز).
4. آدرس `http://localhost/simple-blog/auth/login.php` را در مرورگر باز کنید.

## ورود اولیه (حساب ادمین آماده)

- نام کاربری: `admin`
- رمز عبور: `Admin@123`

## امکانات

- ثبت‌نام و ورود کاربران (با رمز عبور هش‌شده)
- مشاهدهٔ پست‌ها فقط برای کاربران واردشده
- فقط نقش **ادمین** می‌تواند پست بنویسد، ویرایش کند و حذف کند
- کاربر عادی فقط می‌تواند پست‌ها را بخواند
